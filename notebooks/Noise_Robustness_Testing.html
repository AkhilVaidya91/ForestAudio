🔊 Noise Robustness Testing for Forest Audio Classifier¶

This notebook evaluates the robustness of the trained LightGBM (64 MFCC) classifier against various levels of Gaussian noise. The goal is to understand how the model's performance degrades as noise is added to the MFCC feature vectors — simulating real-world noisy conditions at the edge.

Workflow:

  1. Load the trained model (src/model_LightGBM_64.pkl)
  2. Load the test set (data/processed_audio/X_test_64.npy, y_test.npy)
  3. Evaluate baseline (clean) performance
  4. Add Gaussian noise at various SNR thresholds to the test features
  5. Re-evaluate Accuracy, Precision, Recall, and F1 at each noise level
  6. Present results in a summary table and degradation plot

4. Define Noise Injection Utilities¶

We add Gaussian noise to the MFCC features at various Signal-to-Noise Ratio (SNR) levels in dB.
Lower SNR → more noise → harder for the model.

We also test with raw standard-deviation–scaled noise factors for a complementary view.

In [10]:
# Cell 5: Noise injection functions

def add_gaussian_noise_snr(X, snr_db):
    """
    Add Gaussian noise to feature matrix X at a given SNR (dB).
    SNR_dB = 10 * log10(P_signal / P_noise)
    """
    signal_power = np.mean(X ** 2)
    snr_linear = 10 ** (snr_db / 10)
    noise_power = signal_power / snr_linear
    noise = np.random.normal(0, np.sqrt(noise_power), X.shape)
    return X + noise


def add_gaussian_noise_factor(X, noise_factor):
    """
    Add Gaussian noise scaled by noise_factor * std(X) to the features.
    noise_factor = 0 → clean, noise_factor = 1.0 → noise std equals signal std.
    """
    noise = np.random.normal(0, noise_factor * X.std(), X.shape)
    return X + noise


def evaluate_model(model, X, y):
    """Return dict of classification metrics."""
    y_pred = model.predict(X)
    return {
        "Accuracy":  accuracy_score(y, y_pred),
        "Precision": precision_score(y, y_pred, zero_division=0),
        "Recall":    recall_score(y, y_pred, zero_division=0),
        "F1 Score":  f1_score(y, y_pred, zero_division=0),
    }


print("✓ Noise utilities defined")

def highlight_degradation(val):
    """Color-code metric cells: green for ≥0.8, orange for ≥0.6, red below."""
    if isinstance(val, str):
        return ""
    if val >= 0.8:
        return "background-color: #c6efce; color: #006100"
    elif val >= 0.6:
        return "background-color: #ffeb9c; color: #9c5700"
    else:
        return "background-color: #ffc7ce; color: #9c0006"
